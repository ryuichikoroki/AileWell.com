## 2. ページ構成

### 2.1 サイトマップ

```
AileWell.com
├── トップ (index.html)
├── 会社概要 (about.html)
├── 事業・サービス (service.html)
├── ニュース (news.html)
├── 採用情報 (recruit.html)
├── アクセス (access.html)
├── お問い合わせ (contact.html)
└── プライバシーポリシー (privacy.html)
```

### 2.2 各ページの役割

| ページ | 役割 | 主なコンテンツ |
|--------|------|----------------|
| トップ | ファーストインプレッション、導線 | ヒーロー画像、メッセージ、ブロック、LATEST NEWS |
| 会社概要 | 企業理解 | 会社名、理念、事業内容、会社情報 |
| 事業・サービス | サービス紹介 | サービスカード（画像＋説明） |
| ニュース | 情報発信 | ニュース一覧（日付・カテゴリ・タイトル） |
| 採用情報 | 採用訴求 | メッセージ、募集職種、選考フロー、応募方法 |
| アクセス | 所在地案内 | 地図、住所、交通アクセス |
| お問い合わせ | 問い合わせ窓口 | フォーム、連絡先 |
| プライバシーポリシー | 法的記載 | 個人情報の取り扱い |

### 2.3 ナビゲーション

- **ヘッダー：** トップ、会社概要、事業・サービス、ニュース、採用情報、アクセス
- **CTA ボタン：** お問い合わせ（全ページ共通・右端）
- **フッター：** 企業情報リンク、お問い合わせ、プライバシーポリシー

## 3. デザインガイドライン

### 3.1 カラーパレット

| 用途 | 色コード | 用途例 |
|------|----------|--------|
| メインカラー | `#1e90ff` (dodgerblue) | ヘッダー、フッター、見出し、リンク、CTA |
| サブカラー | `#1a7de8` | ホバー、アクセント |
| 背景（薄青） | `#f0f7ff` | ブロック背景 .block-light |
| 背景（白） | `#ffffff` | ブロック背景 .block-white |
| 背景（ダーク） | `#0d1929` | フッター、お問い合わせサイドパネル |
| 本文 | `#1a1a2e` | 見出し・本文 |
| 補足 | `#4a5568` | 説明文 |
| 必須マーク | `#e74c3c` | フォーム必須項目 |

### 3.2 タイポグラフィ

| 用途 | フォント | サイズ例 |
|------|----------|----------|
| 本文 | Noto Sans JP, Yu Gothic, Meiryo | 1rem (16px) |
| 見出し（大） | Noto Sans JP Bold | 2rem〜3rem |
| 見出し（中） | Noto Sans JP Bold | 1.2rem〜1.5rem |
| ラベル | Noto Sans JP Bold | 0.75rem, letter-spacing: 0.2em |

- **Google Fonts：** Noto Sans JP (400, 500, 700)

### 3.3 レイアウト

| 項目 | 値 |
|------|-----|
| 最大幅（コンテンツ） | 880px〜1200px |
| ヘッダー幅 | max-width: 1200px |
| パディング（PC） | 24px〜48px |
| パディング（SP） | 16px〜20px |

### 3.4 ページ別デザイン方針

| ページ | 特徴 |
|--------|------|
| トップ | フルワイドヒーロー、大見出し、ブロックセクション、LATEST NEWS |
| 会社概要 | ページヒーロー、キービジュアル、編集風レイアウト |
| 事業・サービス | カードグリッド、ホバーアニメーション |
| お問い合わせ | 2カラム（フォーム｜連絡先パネル） |
| ニュース | 日付・カテゴリ・タイトルを横並びリスト |
| 採用情報 | キャッチ文、職種カード（左ボーダー） |
| アクセス | 地図エリア、住所カード |
| プライバシー | 狭い幅（640px）、読みやすさ優先 |

## 7. 更新・運用

### 7.1 更新頻度（目安）

| コンテンツ | 頻度 |
|------------|------|
| ニュース | 随時 |
| 採用情報 | 募集開始・終了時 |
| 会社情報 | 変更時 |
| ヒーロー画像 | 必要に応じて |

### 7.2 更新時の注意点

- 画像差し替え：`images/hiro.png` を置き換えるとトップのヒーローが変わる
- ニュース追加：`news.html` の `.news-row` をコピーして追記
- フォーム送信：サーバー側の設定（PHP、Formspree 等）が必要

### 7.3 問い合わせ窓口

- **メール：** info@AileWell.com（実際のアドレスに変更）
- **電話：** 03-XXXX-XXXX（実際の番号に変更）
- **受付時間：** 平日 9:00～18:00
